# AI Agent Rules

- AI may draft, summarize, extract, classify, and propose; it does not get final authority on sensitive publication changes.
- All generated answers must be grounded in approved claims and linked evidence.
- The system must expose uncertainty when support is weak or contradictory.
- Confidence scores must be computed, stored, and surfaced rather than implied.
- Agent actions must record model version, prompt version, and input evidence set.
- High-impact topics must route to human review regardless of model confidence.
