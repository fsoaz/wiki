# Testing Standards

- Validate that each published article has required trust metadata.
- Test claim-to-citation coverage on seeded article fixtures.
- Test contradiction routing with conflicting source inputs.
- Test search answers for citation presence and grounding status.
- Test reviewer workflows for approval, rejection, escalation, and audit output.
- Load-test the read path and hybrid search path independently.
- Run failure tests for LLM outage and source connector degradation.
